            Course Introduction + MATH REVIEW + PAPERS + VIDEOS

            This is an advanced course focusing on fundamentals of Deep Learning. Students will learn about the various neural network architectures like basic Feed-forward Artificial Neural Networks, Recurrent Neural Networks, Convolutional Neural Network, Word Embedding, Self-Attention, Transformers etc. The course will underscore mathematical techniques that enable  deep learning. Practical applications will be illustrated using learned Deep Neural Networks. Understanding Deep Neural Networks and doing well in this class requires at least three things.
            • Prerequisites. This course requires that you have a  good working knowledge of college math, including Trigonometry,  Algebra, Differential Calculus (I and III), Vectors, Matrices and  basic applications you learned in the corresponding classes. Without that  it is very hard to see how and why neural networks work, and then grasping new concepts can become very difficult.
            • Sufficient practice/coding.  Students are expected to do sufficient practice and programming. In this class you will be implementing  neural networks and analyzing code written by others. Please make sure you have enough time to write code.
            • Reading original papers.    This is a graduate level class. Therefore, reading original work of other researchers will be required. We will discuss some important  papers in class together.  You will be expected to study at least one topic and present your findings to the class. It is OK if you do not understand everything in a paper you read. However, you are expected to understand main ideas of that paper and to articulate pros and cons of techniques developed in the paper.

            EXERCISE use the four techniques we discussed for quadratic regression y = a + bx +cx^2:
            • Fix three numbers w=[a,b,c]=[4,2,7].
            • Generate a random vector x of length N (eg N=10, eg x = [2.08299323 2.67247721 3.07063623 3.13688868 3.74846941 3.98480935 4.76398912 5.34806544 5.96618635 6.01499674])
            • Generate random Noise vector of length N
            • Let y = a + bx +cx^2
            • Find optimal parameter values for model fitting NOISY data [x,y]. NOTE if the noise if present the optimal values will be different from [4,2,7]. Use the following techniques:
              • Calculus based method
                • Compute the first derivatves La, Lb, Lc
                • Solve the system of equaitons (SLE):  La=0, Lb=0, Lc=0.
                • You should be able to solve a SLE and arrive at single critical points (CP), which will be your solution minimizing the Loss function L.
                • NOTE: For the calculus based DO NOT ATTEMPT TO DO the 2nd Derivative test. Finding a CP is enough.
              • Vector Projection Method (Use the Normal Equation).
                • Use X =[1, x, x**2]
              • Use GD to minimize the Loss function
              • OPTIONAL: Try to build a NN for solving the same problem. 
            • Compare the results of your solutions.
